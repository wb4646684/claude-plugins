# Jeffrey's Claude Code Plugins

Jeffrey（张锋）自用的 [Claude Code](https://claude.com/claude-code) 插件集合，逐步开放到此仓库。

## 安装市场

```bash
# 在 Claude Code 里
/plugin marketplace add wb4646684/claude-plugins
```

## 可用插件

| 插件 | 说明 |
|------|------|
| [`hap-mcp`](./plugins/hap-mcp) | 明道云 HAP MCP 连接器，含 session token 自动刷新脚本 |

更多插件陆续添加。

## 安装示例

```bash
/plugin install hap-mcp@claude-plugins
```
